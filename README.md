# Smart Bottle Platform

A smart hydration system built for FUSE using a load-cell dock, a bottle-mounted IMU module, and a web app dashboard.

## What it does

- Tracks water intake using a smart dock/base
- Detects sip/tilt behavior with a bottle-mounted IMU
- Shows hydration stats in a web app
- Adds reminders, leaderboard features, and AI-assisted insights

## Project goals

### MVP
- Load-cell dock/base
- IMU bottle module
- Hydration dashboard
- Reminders
- Basic social/leaderboard features

### Stretch goal
- Impeller-based sip-by-sip sensing
